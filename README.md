# TaskMaster Pro

![TaskMaster Pro Logo](https://via.placeholder.com/150)

## Overview
TaskMaster Pro is a modern task management app that helps individuals and teams organize their work efficiently. With smart scheduling features, intuitive reminders, and a clean user interface, TaskMaster Pro ensures you never miss a deadline.

## Features
- Create, edit, and delete tasks with ease
- Smart scheduling that suggests optimal times
- Set priority levels and due dates
- Reminders and notifications
- Task categorization and tagging
- Responsive design for desktop and mobile

## Tech Stack
- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB
- Authentication: JWT
- Deployment: Docker & Heroku

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/rogerskace2/taskmaster-pro.git
   ```
2. Navigate to the project directory:
   ```
   cd taskmaster-pro
   ```
3. Install dependencies for backend and frontend:
   ```
   cd backend && npm install
   cd ../frontend && npm install
   ```
4. Set up environment variables as described in the `.env.example` files.
5. Start the backend server:
   ```
   cd backend && npm start
   ```
6. In another terminal, start the frontend:
   ```
   cd frontend && npm start
   ```

## Usage
Once the app is running:
- Sign up or log in with your credentials.
- Start creating tasks by clicking the "Add Task" button.
- Assign due dates and priorities.
- Organize tasks with tags and categories.
- Enable notifications to get reminders.
